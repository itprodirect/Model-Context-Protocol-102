# Session Logger Rules (Minimal)

This repo uses a simple session log so work can resume quickly.

## Non-negotiables
- Create a new `logs/YYYY-MM-DD_<topic>.md` at the **start** of a session.
- Capture **Decisions**, **Friction**, and **Next — Immediate**.
- End with a **Restart Test**.

## What not to do
- Don’t write essays.
- Don’t log everything.
- Don’t “improve the logger” mid-session—log the friction first, improve later.
